# 🤖 AI PDF Summarizer

An AI-powered Flask web application that summarizes PDF documents using Google's Gemini AI.

## Features

- Upload PDF files
- AI-generated reports
- Page and word count
- Dark Mode
- Multiple Gemini API Key Support
- Responsive UI

## Technologies

- Python
- Flask
- Gemini API
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```
